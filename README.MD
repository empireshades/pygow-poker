## Ultimate Pai Gow

Simulate or play your favorite pai gow poker variant with python!

Created by Jarrod Dowalter

On init (Windows)
`python -m virtualenv venv`

`source venv/Scripts/activate`

`pip install -r requirements.txt`

On init (Mac)
`virtualenv venv`

`source venv/bin/activate`

`pip install -r requirements.txt`
